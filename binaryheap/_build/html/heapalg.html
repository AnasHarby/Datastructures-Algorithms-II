heapalg.py¶

Heap queue algorithms (Priority queues)

A library for the implementation of several heap algorithms.

heapalg.build_heap(heap, reverse=False)[source]¶

Transforms a list into a heap in O(n) time.

Parameters:
  • heap (list) – Already-heapified array.
  • reverse (bool) – A boolean value indicating the reversal of the heap (min_heap -> max_heap).
heapalg.extract(heap, reverse=False)[source]¶

Extracts the root item off the heap, maintaining the heap invariant.

Parameters:
  • heap (list) – Already-heapified array.
  • reverse (bool) – A boolean value indicating the reversal of the heap (min_heap -> max_heap).
heapalg.heapify(heap, n, i=0, reverse=False)[source]¶

Transforms list into a heap in O(n) time, if called with a certain index it assumes that subtrees are already heapified.

Parameters:
  • heap (list) – Already-heapified array.
  • n (int) – Ending index of the array to be heapified.
  • n – Beginning index of the array to be heapified.
  • reverse (bool) – A boolean value indicating the reversal of the heap (min_heap -> max_heap).
heapalg.push(heap, item, reverse=False)[source]¶

Pushes an item onto the heap, maintaining the heap invariant. Assumes heap is already heapified. :type heap: list :param heap: Already-heapified array.

Parameters:
  • item (object) – comparable.
  • reverse (bool) – A boolean value indicating the reversal of the heap (min_heap -> max_heap).
